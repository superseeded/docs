---
title: 'Resolve Specs'
openapi: 'POST /resolve/specs'
---

### Request Body

The request body should be a JSON object with an `items` field containing an array of strings.

<ParamField body="items" type="string[]" required>
  An array of raw plant specification strings to be resolved.
</ParamField>
